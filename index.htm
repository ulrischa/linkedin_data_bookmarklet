How to Install a Bookmarklet

If you want to install the bookmarklet from this page, follow the steps below for your device and browser:

On Desktop

Google Chrome

  1. Make sure the bookmarks bar is visible. Go to Menu (⋮) → Bookmarks → Show Bookmarks Bar.
  2. Click and drag the bookmarklet link to the bookmarks bar.
  3. Done! You can now click the bookmarklet to use it.

Mozilla Firefox

  1. Ensure the bookmarks toolbar is visible. Go to Menu (≡) → Bookmarks → Bookmarks Toolbar → Always Show.
  2. Click and drag the bookmarklet link to the bookmarks toolbar.
  3. Done! You can now click the bookmarklet to use it.

Microsoft Edge

  1. Make sure the favorites bar is visible. Go to Menu (⋯) → Settings → Appearance → Show Favorites Bar.
  2. Click and drag the bookmarklet link to the favorites bar.
  3. Done! You can now click the bookmarklet to use it.

Safari (macOS)

  1. Enable the favorites bar by going to View → Show Favorites Bar.
  2. Click and drag the bookmarklet link to the favorites bar.
  3. Done! You can now click the bookmarklet to use it.

On Mobile

Google Chrome (Android/iOS)

  1. Long-press the bookmarklet link and choose Copy Link Address.
  2. Bookmark any webpage by tapping Menu (⋮) → Bookmark.
  3. Go to Menu (⋮) → Bookmarks, find the newly created bookmark, and tap the Edit button (pencil icon).
  4. Replace the URL with the copied bookmarklet code and save.
  5. Done! You can now tap the bookmarklet to use it.

Safari (iOS)

  1. Long-press the bookmarklet link and choose Copy.
  2. Bookmark any webpage by tapping the Share icon (square with arrow) → Add Bookmark.
  3. Go to Bookmarks, tap Edit, and select the bookmark you just created.
  4. Replace the URL with the copied bookmarklet code and save.
  5. Done! You can now tap the bookmarklet to use it.

Mozilla Firefox (Android/iOS)

  1. Long-press the bookmarklet link and choose Copy Link.
  2. Bookmark any webpage by tapping Menu (≡) → Add to Bookmarks.
  3. Go to Menu (≡) → Bookmarks, find the newly created bookmark, and tap Edit.
  4. Replace the URL with the copied bookmarklet code and save.
  5. Done! You can now tap the bookmarklet to use it.

Note: Always start by opening a webpage before using a bookmarklet, as they often work on the currently loaded page.
